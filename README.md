## Getting Started

Para conseguir executar o projeto, clone o projeto e siga os passos abaixo.

### Instalação

1. You must have the following tools:

- Node.js + NPM
- React Native
- JDK 8
- Android Studio / Genymotion for Emulator

2. Run Commands: 

```sh
yarn install
```

```sh
yarn start --reset-cache
```
**Android**

```sh
yarn android 
```

**IOS**

```sh
yarn ios
```
